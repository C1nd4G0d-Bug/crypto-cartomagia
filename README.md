# 🃏 Crypto Cartomagia (Solitaire Cipher)

Un motor criptográfico implementado en Python basado en el **Cifrado Solitario (Solitaire Cipher)** del criptógrafo Bruce Schneier. Utiliza una baraja de cartas estándar y dos comodines para generar secuencias criptográficas seguras a través de la manipulación física simulada de la baraja.

## 🚀 Características
* Simulación digital de una baraja de 54 cartas (incluyendo Comodín A y Comodín B).
* Algoritmo de mezcla basado en reglas de permutación y cortes (Triple Cut y Count Cut).
* Generación de *keystream* (flujo de claves) para cifrado simétrico.

## 🛠️ Cómo ejecutarlo
```bash
python solitario.py
